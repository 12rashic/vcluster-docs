## Learning objectives

In this guide, you learn the following:

1. How to deploy a virtual cluster.
1. [How to use your virtual cluster](/get-started/02-use-cluster.mdx).
1. [How to configure your cluster](/get-started/03-configure-cluster.mdx).
1. [How to uninstall vCluster](/get-started/04-cleanup.mdx).


You can join the [Loft Users Slack](https://slack.loft.sh/) to ask questions in the #vcluster channel.

Terms:

* **vCluster**: the product.
* `vcluster`: the CLI.
* **host**: The Kubernetes cluster onto which you deploy vCluster. 


## Before you begin

- You have installed [kubectl](https://kubernetes.io/docs/tasks/tools/#kubectl) (check via `kubectl version`) and [Helm v3](https://helm.sh/docs/intro/install/) (check with `helm version`).
- You have access to a Kubernetes cluster to use as the host cluster (check with `kubectl get namespaces`)
- You are familiar with [virutal clusters](index.mdx) and vCluster [architecture](/architecture/README.mdx).
