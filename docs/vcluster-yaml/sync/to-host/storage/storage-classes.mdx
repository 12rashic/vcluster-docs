Sync [StorageClasses](https://kubernetes.io/docs/concepts/storage/storage-classes/) from the virtual cluster to the host cluster. When in enabled and when the necessary CSI driver is installed within the host cluster, certain parameters for the CSI driver will update through synced StorageClasses.

See [Storage](/architecture/storage.mdx) for more information.

### Sync StorageClasses from virtual to host cluster
```YAML
sync:
  toHost:
    storageClasses:
      enabled: true
```
