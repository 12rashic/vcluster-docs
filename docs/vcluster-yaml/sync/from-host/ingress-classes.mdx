Sync IngressClasses from the host cluster to the virtual cluster. This is disabled by default, however, if [ingresses syncing](/docs/vcluster-yaml/sync/to-host/networking/ingresses) is enabled, then this `ingressClasses` option will automatically be enabled as well. Disabling `ingressClasses` can lead to unexpected results if you have enabled `ingresses` syncing.

### Sync IngressClasses from the host cluster to the virtual cluster
```YAML
sync:
  fromHost:
    ingressClasses:
      enabled: true
```
