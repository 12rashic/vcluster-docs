By default, vCluster does not sync IngressClasses from the host cluster to the virtual cluster. However, you can enable it to fully utilize specific features of certain ingress classes available in the host cluster. Without syncing the ingress classes, the virtual cluster ingresses may default to the standard ingress class of the vCluster.

### Sync IngressClasses from the host cluster to the virtual cluster
```
sync:
  fromHost:
    ingressClasses:
      enabled: true
```
