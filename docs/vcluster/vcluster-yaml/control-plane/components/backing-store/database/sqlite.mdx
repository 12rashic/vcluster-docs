By default vCluster uses embedded [SQLite](https://www.sqlite.org/index.html) with a PersistentVolume resource to store data.

You can turn off the PersistentVolume for testing, where losing data on restarts is acceptable.

```yaml
backingStore:
    database:
      embeddedSqlite:
        enabled: true
        dataSource: CONNECTION_STRING
```

Replace _`CONNECTION_STRING`_ with the connection string for your database.
