By default, this is disabled. 

Sync IngressClass resources from the host cluster to the virtual cluster. However, if you enable [ingress syncing](/vcluster/vcluster-yaml/sync/to-host/networking/ingresses), then this `ingressClasses` option is automatically enabled. Disabling `ingressClasses` can lead to unexpected results if you have enabled `ingresses` syncing.

### Enable Ingress and IngressClasses 

Do not enable only IngressClasses.

```YAML
sync:
  toHost:
    ingresses:
      enabled: true
  fromHost:
    ingressClasses:
      enabled: true
```
