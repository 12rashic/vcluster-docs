To upgrade vCluster Platform via vCluster CLI, update `$PLATFORM_VERSION` with a valid vCluster Platform version and run:

```bash
vcluster platform start --upgrade --version=$PLATFORM_VERSION --values=vcluster-platform.yaml
```

:::info Versions and Values
The `$PLATFORM_VERSION` variable above is an environment variable set to your desired vCluster Platform version to
deploy. The `vcluster-platform.yaml` file is an optional YAML file that contains Helm values you would like
to use when upgrading your vCluster Platform deployment.
:::
