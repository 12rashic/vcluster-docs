:::info Support for Manual {props.name}s Only
This only works for {props.name}s without a template, if you want to change the sleep mode configuration for a
    {props.name} created by a template, it must be configured in the template. 
:::

<Flow id="sleep-automatic-ui">
  <Step>
    Click on <NavStep>{props.name === "virtual cluster" ? "Virtual Clusters" : "Spaces"}</NavStep> to find the {props.name} that you want to put to configure automatic sleep mode for.
  </Step>
  <Step>
    Hover over the row of the {props.name} that you want to configure
  </Step>
  <Step>
    Click on <Label>Edit</Label> on the {props.name} that you want to edit. 
  </Step>
  <Step>
    Expand the <Expander>Sleep Mode</Expander> section. 
  </Step>
  <Step>
    Use the <Label>Sleep After Inactivity</Label> field to specify the time to
    wait before putting the {props.name} to sleep if there is no more user
    activity in this {props.name}.
  </Step>
  <Step>
    Click on the <Button>Save Changes</Button> button to save the changes.
  </Step>
</Flow>
