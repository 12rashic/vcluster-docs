Upgrade vCluster Platform via:

<Tabs
  defaultValue="cli"
  values={[
    { label: 'CLI', value: 'cli', },
    { label: 'helm', value: 'helm', },
  ]
}>
<TabItem value="cli">

To upgrade vCluster Platform via vCluster CLI, update `$PLATFORM_VERSION` with a valid vCluster Platform version and run:

```bash
vcluster platform start --upgrade --version=$PLATFORM_VERSION --values=vcluster-platform.yaml
```

</TabItem>
<TabItem value="helm">

To upgrade vCluster Platform via `helm`, update `$PLATFORM_VERSION` with a valid vCluster Platform version and run:

```bash
helm upgrade loft vcluster-platform -n vcluster-platform --repository-config '' --repo https://charts.loft.sh \
  --version $PLATFORM_VERSION \
  --reuse-values \
  -f vcluster-platform.yaml
```

</TabItem>
</Tabs>
